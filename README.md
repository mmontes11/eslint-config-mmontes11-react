Eslint config for ReactJS by Mart√≠n Montes. It extends [airbnb](https://github.com/airbnb/javascript/tree/master/packages/eslint-config-airbnb) and [prettier](https://github.com/prettier/prettier).

### Install

```bash
npm i --save eslint-config-mmontes11-react
```

### Usage

Create an `.eslintrc`:
```
{
  "extends": ["mmontes11-react"]
}
```

### Prettier

You can redefine prettier default rules by creating a `.prettierrc`:
```
{
  "printWidth": 120,
  "trailingComma": "all"
}
```
